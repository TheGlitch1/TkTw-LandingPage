# tk-tw
##Features
  Simple and responsive design
  Tailwind CSS v2 (with PurgeCSS)
  Vue.js v3 with Vue Router
  Theme Switcher with Dark Mode
  Vue.js Smooth Scroll
  Projects Carousel
## Project setup
```
npm install
```
